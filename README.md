# 🌏 Go Together – Travel Across India

A beautifully designed, single-page travel website to explore destinations across India. Built with pure HTML, CSS, and JavaScript — no frameworks required.

## 🖥️ Live Preview

> Open `go-together.html` directly in any browser to run the project instantly.

---

## ✨ Features

- **Hero Section** — Full-screen landing with animated stats (destinations, travelers, states, reviews)
- **Smart Search Bar** — Filter by destination, travel type, and budget
- **Category Filter** — Browse by Mountains, Beaches, Heritage, Wildlife, Spiritual, and more
- **Destination Cards** — 10+ destinations with ratings, prices, best travel time, and highlights
- **Destination Modal** — Click any card for a detailed popup with trip info
- **❤️ Favorites System** — Save favorite destinations (stored in localStorage)
- **Travel Packages** — Budget, Standard, and Premium package cards with inclusions
- **Explore by State** — Visual state cards covering major Indian regions
- **Newsletter Signup** — Email subscription form with validation
- **Sticky Navbar** — Scrolls with shadow effect and mobile hamburger menu
- **Back to Top Button** — Smooth scroll to top
- **Fade-in Animations** — Intersection Observer-based scroll animations
- **Fully Responsive** — Works on mobile, tablet, and desktop

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/go-together.git
   ```

2. **Navigate into the project**
   ```bash
   cd go-together
   ```

3. **Open in browser**
   ```bash
   open go-together.html
   ```
   Or simply double-click the file to open it in your default browser.

> ✅ No build tools, no npm install, no setup needed.
